Texto que se oculta por breackpoints 

            # Las medidas funcionan a partir de un tamaño especifico (breakpoint)
            # Todas estas medidas internamente trabajan con min-width es decir,
            # el tamaño de pantalla debe tener un ancho minimo para que se
            # me empiecen aplicar los estilos, por lo tanto una clase aplicara
            # los estilos mientras el ancho nunca sea menor que el minimo

            Ejemplo:
            Aplicar un margen inferior solo en pantallas muy pequeñas (sm) y que en las
            demas pantallas el margen desaparezca.

            mb-sm-x: para aplicar margenes inferiores en pantallas pequeñas (sm) 
                  donde x es un numero entre 1-5
            
            mb-md-x: para aplicar margenes inferiores en pantallas medianas (md)

            [!] No es necesario definir los otros tamaños de pantallas (lg, xl, xxl) para el
                margen porque con la clase `mb-md-x` aplicara los estilos a partir de un
                ancho minimo de ≥768px (por lo tanto eso incluye tamaños mayores a 768px)

      
            d-flex: a PARTIR de pantallas muy pequeñas    (xs)
            d-sm-grid: a PARTIR de pantallas pequeñas     (sm)
            d-md-block: a PARTIR de pantallas medianas    (md)
            d-lg-none: a PARTIR de pantallas grandes      (lg)
            d-xl-block: a PARTIR de pantallas muy grandes (xl)
